# City Of The Great Machine Statistical Aid App

This is an Android app that provides statistical calculations as an aid to the board game [City Of The Great Machine](https://boardgamegeek.com/boardgame/263895/city-of-the-great-machine) (S&C mode only).


## Background

If you are not familiar with the game, here are the basics you need to understand the App:
>A significant amount of the gameplay is spent deciding where you move your pieces. Once you have chosen your move,
>the 3 servants (pieces controlled by the game itself) each move. Each one draws a card from the movement deck and moves
>somewhere on the board depending on what card was drawn. If one of the servants moves to the same place as one of your
>pieces, that causes a detainment. A major part of the strategy to consider the cards remaining in the movement deck
>and what will happen if each one is drawn. If there are too many cards that could cause detainment(s) you start to 
>consider alternate plans. This app will be tremendously useful in this regard, because it gives you an actual percent
>chance that a detainment will occur.


## Purpose

This app helps players better understand the probabilities of success and failure in the game, aiding strategic decision-making 
for an improved gameplay experience.


## Usage

- Input the cards in the movement deck.
- Enter the number of detainments each card would cause if drawn.
- Calculate the probability of detainment.
- Get a full breakdown of the chances and the number of detainments.
